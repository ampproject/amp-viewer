# amp-viewer


## Developing

| Command                                                                 | Description                                                           |
| ----------------------------------------------------------------------- | --------------------------------------------------------------------- |
| **`yarn build`** | Runs "build" |
| **`yarn watch`** | Runs "build" with watch and runs a localhost webserver |
| **`yarn test`** | executes tests in /tests directory |
| **`yarn clean`** | deletes the /dist directory |
